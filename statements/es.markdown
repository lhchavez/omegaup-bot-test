# Descripción

Este es un problema de prueba. Puedes usarlo para familiarizarte con el ambiente de omegaUp.

# Entrada

Dos números enteros, $a$ y $b$.

# Salida

La suma de ambos números.

# Ejemplo

||input
1
2
||output
3
||description
1 + 2 = 3
||input
5
10
||output
15
||description
5 + 10 = 15
||end

# Restricciones

* $-10^{18} \le a, b \le 10^{18}$
